**Number Guessing Game**

_Overview:_

_This project is a simple Python-based Number Guessing Game._

_The user first selects a lower bound and an upper bound for the range of numbers. The system then generates a random number within that range,
and the user gets 5 attempts to guess the correct value._

_After every guess, the system gives feedback:_

_Too low_
_Too high_
_Correct_

_At the end of the game:_

_1 is displayed if the user wins_

_0 is displayed if the user loses_
